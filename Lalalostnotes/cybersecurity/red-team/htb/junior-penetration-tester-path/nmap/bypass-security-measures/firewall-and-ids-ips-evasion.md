# Firewall and IDS/IPS Evasion

#### **Perbandingan SYN Scan vs ACK Scan**

| Fitur          | SYN Scan (`-sS`)                     | ACK Scan (`-sA`)                      |
| -------------- | ------------------------------------ | ------------------------------------- |
| **Tujuan**     | Ngecek port open/closed              | Ngecek filtering firewall             |
| **Cara Kerja** | Kirim SYN, terima SYN-ACK, kirim RST | Kirim ACK, terima RST (kalo filtered) |
| **Kecepatan**  | Cepat                                | Cepat                                 |
| **Stealth**    | Tinggi (nggak nyelesaiin handshake)  | Sedang (nggak ngecek port open)       |
| **Hasil**      | Tau port open/closed                 | Tau port filtered/unfiltered          |
